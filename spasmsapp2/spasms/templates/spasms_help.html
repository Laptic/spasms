To use the software

1.On the main page, create an exercise, you can create as many xercises as you want.

  • To clarify, "Percent female" is used to specify how many people in "Num users" is female.

2.On the main page, create a run, you can create as many runs in a given Exercise as you want.

3.On the main page, you can view all the exercises and runs that were created with each run.

  • In the runs list, you are able to export the data of a given run.

4.On the main page, you can view the exported data from the previous step
