How to use the application

The main purpose of this application is to automatically generate tweets about a specified topic and with a specified sentiment.

  • On the main page, you have the option to create a new exercise, and you will be able to view all the exercises you have created.
  • Once you have created an exercise, you will have the option to create a new group for thata exercise. In order to be able to create a run to generate tweets, you must have created at lease one group in the exercise.
    • The "Percent female" field is used to determine what percent of users specified by "Num users" will be female
    • You can also create a group from the main page, by clicking on the create group button next to the exercise iin which you would like to create a group.
  • Once you have created a group, you will have the option to create a run for that group.
    • In the form for creating a run, you will need to specify a sentiment and a noun to be used for the tweets generated
    • When you submit the form to create a run, all the tweets will be generated and stored to the database, and the tweet data for that run will also be exported to json file
  • After creating a run, you will have the option to view the json data which will show all the tweets generated in that run.
  • At any point while using the application, you can click on Welcome in the top navigation bar to return to the home page.
  • To view all the groups in an exercise, you can expand the exercise from the main page and click on "View more info."
  • From the groups page, you can view the group's informatino by expanding each group. After expanding a group, you will have the option to view all the runs for that group, or view all the users in that group.
  • In the main page, the groups page, and the runs page, you will have the option to delate an exercise, a group, and/or a run.
    • When you delete an exercise, all the groups, users, runs, and tweets associated with that exercise will also be deleted
    • When you delete a group within an exercise, all the users associated with that group and all the runs and tweets generated for that group will also be deleted
